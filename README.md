# Welcome to Gorganisation !

Gorganisation is a sandbox app where I have fun and sharpen my skills with:
- Go
- Buffalo (http://gobuffalo.io)
- React
- GraphQL (https://github.com/samsarahq/thunder).

It is a simple page that displays a random wallpaper of the day from https://unsplash.com, a quote of the day from http://quotes.rest and a todo list coming from the buffalo back end.
